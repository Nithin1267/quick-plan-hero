# Standalone HTML version

A single-file Smart To-Do List — pure HTML, CSS, and JavaScript. No build step, no server.

## Run locally
Just open `index.html` in your browser.

## Deploy to GitHub Pages
Because this folder is fully static, it **does** work on GitHub Pages (unlike the main TanStack Start app at the repo root).

1. Push the repo to GitHub (Lovable syncs automatically when connected).
2. In your repo: **Settings → Pages**
3. **Source:** Deploy from a branch → **Branch:** `main` → **Folder:** `/standalone`
4. Save. Your site will be live at `https://<user>.github.io/<repo>/`.
